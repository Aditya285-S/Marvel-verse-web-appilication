## Commands to run application

- ### Create Environment
  
  ```bash
  virtualenv 'enter environment name'
  ```
  - Eg.myenv

- ### Activate environment

  ```bash
  .\myenv\scripts\activate
  ```

  ```bash
  pip install requiremnets.txt
  ```

  ```bash
  cd marvelverse
  ```

  ```bash
  python manage.py runserver
  ```
